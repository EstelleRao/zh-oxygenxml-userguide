发布DITA Map

使用 Oxygen XML Author 发布 DITA 文件（DITA Topic 或 DITA Map），需要先配置发布场景。在发布场景中添加 CSS 样式表，可将 DITA 文件导出为定制的 PDF 样式。在发布场景中添加过滤文件，可根据过滤条件导出指定的说明书。

请按照如下步骤配置 DITA Map 的发布场景，发布 DITA Map。

  1. 在 DITA Maps Manager 视图打开并选中 index.ditamap。
    Tip:
    • 如需发布 DITA Topic（指定的小节），则在编辑区域打开DITA文件，将光标定位在文件内的任意位置。
  2. 点击 Configure Transformation Scenario(s)，在 Configure Transformation Scenario(s) 窗口选择 DITA Map PDF - based on HTML5 & CSS，创建 DITA Map PDF 发布场景。
  3. 点击 Edit，配置发布场景。
    1. 在 Templates 标签页，点击 Choose Custom Publishing Template，选择 .opt 发布模板。
    2. 在 Parameters 标签页，将光标定位在参数列表上方的搜索框，分别输入图表交叉引用参数和位置参数，设置图表的交叉引用格式和标题位置。
    3. 在 Filters 标签页，点击 Use DITAVAL file，选择指定说明书对应的过滤文件。如果不添加过滤文件，则默认导出整个 index.ditamap 为一本说明书。
      Tip:
      • 如需导出多本说明书，请使用批量发布文件（cornerstonepdfpublishing_{product}.xml）。
  4. 点击 Apply associated 发布说明书。
  5. 发布后，检查说明书样式。
